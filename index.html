Bienvenue sur le site de gestion des câbles et appareils

Ce site est un prototype conçu pour aider à la gestion des câbles et des appareils sur le navire , en se basant sur le NEC (Nomenclature des Equipements de Câblage). Pour l’instant, les données sont à jour manuellement — une version auto-maj pourrait suivre.

Objectifs du site

Le site permet d'accéder à différentes fonctionnalités :

  • Onglet Câble : chercher un câble précis + voir ses infos.
  • Onglet Appareil : chercher un appareil + voir ses câbles liés.
  • Onglet R&D : tester plein d'appareils à la fois (colle une colonne Excel), voir leur statut, exporter ce qui reste à tirer ou ce qui est OK pour branchement.
  • Onglet Générer JSON : convertir un NEC Excel en JSON utilisable par le site.

Les données sont celles du navire , mais tu peux adapter le site à un autre navire (même format de NEC) en changeant `config_changement_navire.js`. Un MODOP est prévu pour expliquer ça proprement.
